# Qtile Keyboard Shortcuts

**Modifier Key:** `Mod` = Super/Windows key

| Category           | Shortcut                   | Action                         |
| ------------------ | -------------------------- | ------------------------------ |
| **Window Focus**   | `Mod + h/j/k/l`            | Move focus left/down/up/right  |
|                    | `Mod + Space`              | Move focus to next window      |
| **Window Move**    | `Mod + Shift + h/j/k/l`    | Move window left/down/up/right |
| **Window Resize**  | `Mod + Ctrl + h/j/k/l`     | Grow window left/down/up/right |
|                    | `Mod + n`                  | Reset all window sizes         |
| **Window Control** | `Mod + w`                  | Kill focused window            |
|                    | `Mod + f`                  | Toggle fullscreen              |
|                    | `Mod + t`                  | Toggle floating                |
|                    | `Mod + Shift + Return`     | Toggle split/unsplit stack     |
| **Layout**         | `Mod + Tab`                | Toggle between layouts         |
| **Workspaces**     | `Mod + 1-0`                | Switch to workspace 1üè†-0üîß    |
|                    | `Mod + z`                  | Toggle to last used workspace  |
|                    | `Mod + Shift + 1-0`        | Move window to workspace       |
| **Screens**        | `Mod + .`                  | Focus next screen              |
|                    | `Mod + Shift + .`          | Move workspace to next screen  |
|                    | `Mod + Shift + m`          | Monitor configuration menu     |
| **Launchers**      | `Mod + Return`             | Launch terminal                |
|                    | `Mod + r`                  | Launch rofi                    |
|                    | `Mod + grave`              | Toggle btop popup              |
|                    | `Mod + Shift + /`          | Toggle help popup              |
| **System**         | `Mod + Ctrl + r`           | Reload qtile config            |
|                    | `Mod + Ctrl + q`           | Shutdown qtile                 |
|                    | `Mod + Alt + l`            | Lock screen                    |
|                    | `Mod + Shift + e`          | Power menu                     |
|                    | `Mod + Shift + p`          | Screenshot menu                |
|                    | `Mod + Shift + n`          | Notification history           |
|                    | `Mod + F1`                 | Test key binding               |
| **Audio**          | `XF86AudioMute`            | Toggle audio mute              |
|                    | `XF86AudioRaise/Lower`     | Volume up/down 5%              |
|                    | `F20`                      | Toggle microphone mute         |
| **Brightness**     | `XF86MonBrightnessUp/Down` | Screen brightness              |
|                    | `XF86KbdBrightnessUp/Down` | Keyboard backlight             |
| **Media Keys**     | `XF86Launch3`              | App menu (rofi drun)           |
|                    | `XF86Launch4`              | Window switcher                |
| **Mouse**          | `Mod + Left Drag`          | Move floating window           |
|                    | `Mod + Right Drag`         | Resize floating window         |
|                    | `Mod + Middle Click`       | Bring window to front          |
